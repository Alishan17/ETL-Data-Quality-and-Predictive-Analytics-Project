# ETL-Data-Quality-and-Predictive-Analytics-Project
Built an end-to-end data cleaning and analysis pipeline for a 20,000+ record retail sales dataset using Python and SQL.

Performed data quality validation to identify foreign key violations, inconsistent IDs, and rule-based errors. Removed 397 invalid records and ensured referential integrity before analysis.

Created a transformed analytics dataset and explored revenue trends across stores and employees using summary statistics and visualizations.

Developed a linear regression model to predict revenue based on effort hours. Found a strong positive relationship, with revenue increasing by ~27.4 per additional hour of effort.

Key skills demonstrated:

- Data cleaning & validation
- SQL-based integrity checks
- Exploratory data analysis (EDA)
- Data visualization (Matplotlib)
- Predictive modelling (scikit-learn)

Tools: Python, Pandas, DuckDB, SQL, Matplotlib, Scikit-learn, Jupyter Notebook
